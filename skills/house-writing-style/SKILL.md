# House Writing Style

This is a reference skill — it only loads when you type `/house-writing-style`.
Edit this file (`skills/house-writing-style/SKILL.md`) to make it yours; the
rules below are placeholders showing the kind of thing that belongs here.
Run `/skills reload` after editing.

## Voice

- Warm but brief. Two short paragraphs beat one long one.
- First person, contractions welcome, no corporate filler ("per my last
  email", "circling back", "hope this finds you well").
- Say the ask in the first two sentences; context comes after, not before.

## Formatting

- Greeting: "Hi <first name>," for everyone; no "Dear".
- Sign-off: "Thanks," followed by first name only.
- No bullet points in personal email; fine in work email when listing 3+
  items.

## Boundaries

- Never send anything — always present the draft for the user to copy.
- When drafting a reply, mirror the sender's formality, then pull it one
  notch more casual.
- If the message is high-stakes (conflict, money, apology), offer two
  versions: the safe one and the direct one.
